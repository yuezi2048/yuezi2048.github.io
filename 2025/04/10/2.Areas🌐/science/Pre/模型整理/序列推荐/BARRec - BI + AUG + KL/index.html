BARRec - BI + AUG + KL

Improving Sequential Recommendations via Bidirectional Temporal Data Augmentation with Pre-training

序列推荐系统对于识别用户的时间性偏好至关重要。然而，从简短的用户交互序列中进行学习这一任务面临着显著挑战。数据增强已被视为一种能够增强这些序列信息丰富度的有效策略。传统的数据增强技术，比如物品随机化，可能会破坏序列中固有的时间动态性。尽管最近在按逆时间顺序生成伪物品方面取得的进展展现出了潜力，但在自然的时间顺序背景下进行评估时，这些方法可能会引入时间上的差异。 对此，我们提出了一种先进的方法——基于预训练的双向时间数据增强（BARec）。我们的方法利用双向时间增强和知识增强微调，来合成真实的伪先验物品，这些物品既能保留用户偏好，又能捕捉到更深入的物品语义关联，从而提升模型的表达能力。我们在五个基准数据集上进行的全面实验分析证实了，无论是在短序列还是长序列的情境下，BARec都具有优越性。此外，理论分析和案例研究为深入理解该模型的推理过程和可解释性提供了更多的见解。我们这项研究的源代码可在https://github.com/juyongjiang/BARec上公开获取。

![[Pasted image 20250410113029.png]]

  1. 模型无关主干网络模块( CMHSA + PFFN）
    • 嵌入层：由物品嵌入和位置嵌入组成。物品嵌入将离散的物品标识符映射到连续的dmodel​维潜在空间，位置嵌入矩阵则用于捕捉序列中物品的位置信息。两者结合，把输入序列映射到联合嵌入空间，从而有效捕获物品特定特征及其在序列中的位置上下文 。
    • 编码器层：包含因果多头自注意力（CMHSA）和位置全连接前馈网络（PFFN）两个子层。CMHSA 通过多头自注意力机制和因果掩码策略，结合缩放点积注意力及层归一化，捕捉序列中的上下文模式；PFFN 由一对线性变换和 ReLU 激活函数构成，进一步细化序列中每个位置的嵌入表示，编码更复杂的特征 。
  2. 双向时间数据增强与预训练模块（BI + AUG）
    • 伪先验物品生成：将用户特定序列反转输入到编码器中，通过预训练模型进行自回归预测，生成伪先验物品。在这个过程中，通过双向时间预训练，确保生成的伪先验物品与用户正向偏好一致，增强物品间的语义联系。
    • 预训练损失函数：在预训练阶段，构建伪先验物品标签，通过最小化预测这些标签的损失来训练模型。同时，为了使生成的伪先验物品更符合用户偏好，在损失函数中增加了基于用户偏好的部分，平衡正向预测和反向生成 。
    • 自回归物品生成：根据基准数据集的序列长度分布特点，引入超参数 κ 和 M，分别表示要生成的伪先验物品数量和序列长度阈值。通过自回归生成方式，对短序列进行数据增强，扩充序列数据。
  3. 知识增强微调模块( KL )
    • 微调损失函数：利用增强后的序列数据对预训练模型进行微调，采用修改后的常见的移动一步标签技术，选择增强序列数据中最近的一部分真实标签进行训练，以解决短序列和冷启动问题。
    • 模型知识增强：由于数据增强可能导致模型过拟合等问题，采用双向 Kullback-Leibler（KL）散度来对齐增强数据和原始数据的概率分布，实现知识增强。将这一策略集成到微调损失函数中，平衡下一项预测准确性和知识增强
  4. 黄金标签裁剪操作（LC） 能解决短序列长度带来的限制，为短序列信息稀缺问题提供部分解决方案
