补充

  • API 差异：TensorFlow 和 PyTorch 的 API 在某些方面有所不同，需要仔细对照文档进行调整。

  • 数据类型：TensorFlow 使用 tf.Tensor，而 PyTorch 使用 torch.Tensor，注意数据类型的转换。

  • 损失函数和优化器：虽然两者都有类似的损失函数和优化器，但具体实现和参数可能有所不同。

  • 设备管理：PyTorch 默认使用 CPU，需要显式地将数据和模型移动到 GPU 上（使用 .to(device)）。
