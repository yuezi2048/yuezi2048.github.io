43.基本数据类型

发表于2025-04-24|更新于2025-04-24|2.Areas🌐02.面经记录面试鸭Java基础
|总字数:281|阅读时长:1分钟|浏览量:

Java有8种基本数据类型

整型

  • byte 1字节，-128~127
  • short 2字节，-32768~32767
  • int 4字节，-2^31~2^31-1（2147483648）
  • long 8字节，-2^63~2^63-1

浮点型

  • float 4字节
  • double 8字节

字符型

  • char 2字节，存储单个Unicode字符，范围0~65535

布尔型

  • boolean 存放true和false，具体大小依赖JVM实现

拓展

基本数据类型的特性：

  • 大小固定：保证跨平台的一致性
  • 不支持null：声明时会有默认值，int为0，boolean默认为false
  • 性能高：基本类型直接存在栈中，效率高于包装类Integer、Double等

默认值：

整型0，浮点型0.0，字符型'\u0000'，布尔型false

类型转换：

  • 隐式转换：小类型到大类型，不会数据丢失，如int-->long
  • 显式转换：大类型到小类型，可能精度丢失或溢出，如long->int或double->float

boolean存储空间

  • 逻辑上boolean是占用1位，但JVM对boolean存储会根据系统架构会对boolean分配1字节或更多位数的空间，因为CPU是按字节操作内存，而不是按位
