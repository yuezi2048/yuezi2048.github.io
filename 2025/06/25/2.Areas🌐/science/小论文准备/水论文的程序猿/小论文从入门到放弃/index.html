小论文从入门到放弃

发表于2025-06-25|更新于2025-06-25|2.Areas🌐science小论文准备水论文的程序猿
|总字数:1.3k|阅读时长:4分钟|浏览量:

看哪些论文

不用看的：普刊、中文核心、ei会议、ei源刊（sci四区）

（发ei会议可以参考一下）

要看的：SCI（SSCI）四个级别，CCF会议（三个级别）

  • 二三区（CCFB）没必要看，不上不下
  • 看最好的（一区，CCFA）和最烂的（四区，CCFC）
    • 一区（顶刊）：关注吹牛皮的方法
    • 四区：赤裸裸A+b+c 的看的很清楚，明确学术裁缝的方法

读论文的方法

  • 不用精读，挑重点

摘要

引言（抄）+ 相关工作（抄） + 方法概括（重点看） + 实验效果

要点：按部分（标点符号，转折词等断句）来读英文，对我们有用的再精读

方法：CLIP + DCD

引言

基于…背景，做了…工作，这个工作….在做，基于…做的，提出了…问题，然后提出了…方法解决这个问题，实验效果还不错。

问题，效果等部分，加以图来描述

启发式写作手法：基于xx问题使用了xxx基准模型，基于..问题又提出了..模块

相关工作（不要看）

瞎扯的。后面写论文需要参考

引入的模型，一般模型是同领域的，被别人写烂了，写的时候把内容直接搬过来，写作方式重构即可（语句，参考文献替换）

间章：和传统方法的比较

传统的两个工作量：A + b + c，而该论文是在前任提出了B的基准模型的基础上，加上了探讨和传统的A的比较，来当作一个模块

3.2 基准模型说明

实验参数：只在MSR-VTT数据集上讨论，大数据集性能差的不说

分析部分：使用提问的方式，阐述为什么用A+b，除了用表格佐证，挑一个有利的数据，模仿同领域的画图，来佐证

方法

客观事实陈述模型

基准模型的概述

4.1（3.3） 公式（抄）

4.2 （3.4） DCD介绍

4.3 实验参数，写的时候抄

4.4 对比实验（和实验章节分开了），和近几年的sota对比，看的时候没必要看，对比的时候可以挑比自己差的，但是要新

4.5 消融实验（和实验分开了），去掉DCD，只加视频的CCD，只加文本的TCD，全加的DCD，用图式的会更加生动（Qualitative results）

实验与分析（不要看）

佐证自己方法的

总结

方法 + 实验总结，或者方法 + 实验效果

关注和摘要的区别：摘要要说为什么要做这些事情，总结是说明用了什么方法，以及方法做成的效果，直抒胸臆即可

参考文献

  • 水的概念做一个参考
  • 相关工作加的多
  • 对比实验的参考文献是有意义的，大概率这篇论文的A就在这里面

顶刊实例

摘要

  • 故事：文本和视频的间隙造成的表达问题
  • 方法：A + b + c，VTM + Transformer + 集成策略
  • 效果上说明了公有数据集的参数提升10%，这里额外说明了竞赛上的私有数据集

拆分出来，对于方法理论的介绍一定是按照这个顺序介绍的

引言

看段落在说什么

  • 领域介绍
  • 领域发展史
  • 这样的发展出现的问题，提出了CLIP
  • 说明CLIP的A + b’ + c，发现b的解码过程做了某种改进

理论

A = VTM = CLIP + TSN采样，改进：CLIP + AAA采样

B’ 由于大数据集，transformer进行了层数下降处理 缓解过拟合

C: 每个指标加权平均的集成策略，那我改进，做一个w1，w2训练一个网络改进

实验分析

结论

CLIP：A + b’ + C，效果

缝模块

通过大量的阅读，找到共通点（相似的基准模型）

含有共通点A的论文里面确定一篇论文作为基准模型，把A作为基准也可以，一定要找代码可复现的论文

  • CapVideo（A）= CLIP + TVM + DAAC + VCCMI
  • CLIP+DCD
  • CLIP+TSN+Uni_VL+ES
  • CLIP+ABC+EDF+GHI
  • INP + Mo + AGH

实验1：CYD4 = Cap4（CLIP）+DCD’+TSN’

实验2：CLIP+DCD‘+AGH’

(Cap4 - DAAC) + EDF’ + ES’ + DCD’ + MO’

通过实验去做排列组合，并确定模型的有效性，性能有提升

写论文

方法

首先写方法，（参考顶会确定架构图的位置），通过数据流的方式讲清楚

  • 按模块介绍（可以模仿别人的架构）：
    • 3.1 a c d 主体
    • 3.2 b
    • 3.3 训练过程
  • 最终目的：让别人听懂方法

注：方法没有固定的范式，关键是确定模型，如何划分模型，然后分开展开写（总-分-分-分）

  • 细节部分，对应论文的介绍 重构（目标期刊同领域论文 最好不过）

实验

  • 4.1 数据集介绍 抄
  • 4.2 实验设置
  • 4.3 实验分析
    • 对比实验
      • 找5-10个对比模型（1篇25年 2篇24年 3篇23年）， 抄
    • 消融实验：佐证 b’和c‘的有效性，花1~2句话编一个故事，如何提高了性能。
      • w/o c’
      • w/o b’
      • w/o b’ + c’
      • no w/o
    • 实例分析 case study
      • 用不同的模型，找一个对应的推荐，挑出最好的3~5个，可视化结果分析
