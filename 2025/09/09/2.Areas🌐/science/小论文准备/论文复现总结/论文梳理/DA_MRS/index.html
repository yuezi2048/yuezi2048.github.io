DA_MRS

当前领域存在的问题

多模态推荐系统（MRS）面临三个主要问题：

  1. 多模态内容噪声：物品的图片、文本等描述可能包含无关信息（例如商品图片背景干扰或文本描述冗长），导致物品相似性计算不准确。
  2. 用户反馈噪声：用户点击、浏览等行为可能包含误操作或偏见（例如随机点击或受广告影响），误导模型学习。
  3. 多模态内容与用户反馈的对齐不足：现有方法仅通过对比学习对齐不同模态的表示，但忽略了用户偏好视角的细粒度对齐（例如相似物品的层级关系）。

现有方法

现有方法主要分为两类：

  1. 基于特征的方法（如VBPR、MMGCN）：直接融合多模态特征增强物品表示，但未显式处理噪声。
  2. 基于结构的方法（如LATTICE、MICRO）：通过多模态内容构建物品相似图，但存在以下问题：
    • 未过滤跨模态不一致的相似性（例如图片相似但文本无关的物品被错误连接）。
    • 未利用用户行为数据补充物品关系。
    • 对齐方法仅停留在物品级别，未考虑用户偏好和细粒度相似性。

本文提出的方法：DA-MRS

DA-MRS包含三个核心模块，分别解决上述问题：

1. 去噪物品-物品图（Denoising Item-item Graph）

  • 方法步骤：
    • 跨模态一致性过滤：计算每个模态的相似矩阵，删除与其他模态不一致的相似项（例如仅图片相似但文本不相似的边）。
    • 行为图补充：基于用户共现行为构建物品关系图，弥补内容噪声导致的稀疏性。
  • 作用：生成更准确的物品关系图，避免噪声传播。

2. 去噪用户反馈（Denoising User Feedback）

  • 方法步骤：
    • 概率化BPR损失：将用户反馈的可靠性建模为伯努利分布：
      • 正样本置信度（f(u,i)）：用户对物品多模态内容的偏好一致性（跨模态均值+方差）。
      • 负样本置信度（g(u,i))：用户因单一模态吸引或缺乏替代品而误点击的概率。
  • 作用：降低噪声反馈的权重，增强可靠信号的监督作用。

3. 多模态内容与用户反馈对齐（Alignment）

  • 用户偏好对齐（AU）：最小化用户基于多模态内容与基于行为的偏好分布差异（KL散度）。
  • 层级物品关系对齐（AI）：
    • 将物品分为三类对比样本：全模态相似、单模态相似、不相似。
    • 通过对比学习拉近全模态相似物品，推开不相似物品。
  • 作用：实现用户视角的细粒度对齐，提升推荐相关性。

创新点如何解决问题

  1. 跨模态一致性过滤：通过多模态交叉验证，减少因单一模态噪声导致的错误连接。
  2. 概率化BPR：利用多模态内容估计用户真实偏好，区分可靠与噪声反馈。
  3. 层级对齐：通过细粒度对比学习，更好区分物品的相似程度（例如“同款不同色”vs“同类不同款”）。

实验效果

在Baby、Sports等数据集上，DA-MRS显著优于基线模型（如Recall@20提升7.5%~23.8%），且在噪声场景下表现鲁棒。

模块组成

LightGCN’（去噪物品图） + 概率化BPR损失（反馈去噪） + 用户偏好对齐（AU） + 层级物品关系对齐（AI）

以下是针对论文《Improving Multi-modal Recommender Systems by Denoising and Aligning Multi-modal Content and User Feedback》的去噪项目（Denoising Item-item Graph）的步骤总结，包含公式和完整流程：

多模态内容与用户反馈对齐的步骤总结

1. 多模态内容去噪（Denoising Multi-modal Content）

目标：减少多模态内容（如图像、文本、音频）中的噪声，构建更准确的物品-物品关系图。
方法：

  1. 计算模态内相似度矩阵：
    • 对每个模态 ( m )，计算物品 ( i ) 和 ( j ) 的余弦相似度：
      [
      S^m_{i,j} = \frac{(e^m_i)^T e^m_j}{|e^m_i| |e^m_j|}
      ]
    • 过滤低相似度边（( S^m_{i,j} < \overline{S^m} ) 时置零）。
  2. 跨模态一致性过滤：
    • 若某边在任一模态中被过滤（( S^{m’}_{i,j} = 0 )），则在所有模态中移除该边。
  3. 构建物品-物品语义图（IIS-Graph）：
    • 对每个物品 ( i )，保留 top-( K ) 相似物品构建邻接矩阵 ( A^m )（公式1）。

公式：
[
A^m_{i,j} = \begin{cases}
1, & \text{if } S^m_{i,j} \in \text{top-}K(S^m_{i,:}) \text{且} S^m_{i,j} > 0 \
0, & \text{otherwise}
\end{cases}
]

2. 用户反馈去噪（Denoising User Feedback）

目标：利用多模态内容估计用户反馈的可靠性，修正噪声交互。
方法：

  1. 定义反馈生成概率：
    • 用户 ( u ) 对物品 ( i ) 的反馈可靠性 ( f(u,i) ) 由多模态偏好的一致性决定：
      [
      f(u,i) = (\mu_{ui})^\alpha \times (e^{-s^2_{ui}})^\beta
      ]
      其中 ( \mu_{ui} ) 为跨模态偏好均值，( s^2_{ui} ) 为方差。
    • 噪声反馈概率 ( g(u,i) ) 由单模态最大偏好与负样本均值决定（公式7）。
  2. 去噪BPR损失（D-BPR）：
    • 结合 ( f(u,i) ) 和 ( g(u,i) ) 优化模型：
      [
      \mathcal{L}{\text{D-BPR}} = \sum{\langle u,i,j \rangle} \ln \left[ f(u,i) \sigma(u_u^T(t_i - t_j)) + g(u,i) (1 - \sigma(u_u^T(t_i - t_j))) \right]
      ]

3. 多模态内容与用户反馈对齐（Alignment）

目标：增强任务相关的物品表示，细化物品关系对齐。

3.1 用户偏好引导对齐（Alignment guided by User Preference）

步骤：

  1. 计算用户 ( u ) 基于多模态内容 ( h^{mm}_i ) 和ID嵌入 ( h^{id}_i ) 的偏好分布 ( P^{mm}_u ) 和 ( P^{id}_u )（公式9）。
  2. 最小化两者KL散度：
    [
    \mathcal{L}{\text{AU}} = \sum{u} \left[ \text{KL}(P^{mm}_u | P^{id}_u) + \text{KL}(P^{id}_u | P^{mm}_u) \right]
    ]
3.2 分级物品关系对齐（Alignment guided by graded Item Relations）

步骤：

  1. 分级正负样本：
    • 多模态相似物品（( R^m_i )）：跨模态相似度最高的物品。
    • 单模态相似物品（( T^m_i )）：仅当前模态相似的物品。
    • 不相似物品（( N^m_i )）：随机负样本。
  2. 对比学习损失：
    [
    \mathcal{L}{\text{AI-MM}} = -\log \frac{\sum{j \in R^m_i} \phi(h^m_i, h^m_j)}{\sum_{j \in R^m_i} \phi(\cdot) + \sum_{k \in T^m_i} \phi(\cdot) + \sum_{l \in N^m_i} \phi(\cdot)}
    ]
    其中 ( \phi(h_i, h_j) = \exp(\text{sim}(h_i, h_j)/\tau) )。

4. 总损失函数

[
\mathcal{L} = \mathcal{L}{\text{D-BPR}} + \lambda_1 \mathcal{L}{\text{AU}} + \lambda_2 (\mathcal{L}{\text{AI-MM}} + \mathcal{L}{\text{AI-S}})
]

关键贡献总结

  1. 去噪：通过跨模态一致性过滤噪声，构建更鲁棒的物品关系图。
  2. 对齐：
    • 用户偏好对齐增强任务相关表示。
    • 分级物品关系对齐细化相似度学习。
  3. 通用性：可插拔到多种骨干推荐模型（如LightGCN、MF）。

实验表明，DA-MRS在噪声场景下显著优于基线方法（如LATTICE、SLMRec），尤其在稀疏和噪声数据中表现稳健。

主要实验结论与对应图表

  1. DA-MRS的总体性能优势（RQ1）

    • 结论：DA-MRS在多个数据集（Baby、Sports、Clothing、TikTok）上显著优于传统推荐系统（RS）和多模态推荐系统（MRS），平均提升幅度达7.49%~23.83%（见表2）。
    • 对应图表：
      • 表2：对比了DA-MRS与基线模型（如MF-BPR、LightGCN、LATTICE等）的Recall@20、Precision@20和NDCG@20指标，显示DA-MRS在各类骨干模型（MF、VBPR、LightGCN）上均取得最优效果。
      • 关键提升：在Clothing数据集（稀疏性最高）上提升最显著（+187% Recall@20），验证了多模态内容对稀疏反馈的补充作用。

  2. 抗噪性能验证（RQ2）

    • 多模态内容噪声：
      • 结论：当人为添加噪声（替换5%~20%的多模态特征）时，DA-MRS性能下降最小（图3）。例如，在20%视觉噪声下，DA-MRS的Recall@20仍优于无噪声时的基线模型。

    • 用户反馈噪声：
      • 结论：在随机添加或删除反馈数据（5%~20%）的场景下，DA-MRS表现最稳定（图4），删除反馈时性能下降率仅9%，远低于基线。
      • 对应图表：
        • 图4(a)(b)：对比不同噪声比例下DA-MRS与基线模型的Recall@20，DA-MRS抗噪性最强。

  3. 模块消融实验（RQ3）

    • 关键模块贡献：
      • Denoising Item-item Graph (DIIG)：去除跨模态不一致的相似性链接，提升基础性能（表3中DIIG比LightGCN提升23.8% Recall@20）。
      • Denoised BPR Loss (D-BPR)：通过多模态内容估计反馈置信度，进一步优化训练（表3中DIIG+D-BPR比DIIG略优）。
      • Alignment模块：
        • Alignment guided by User preference (AU)：对齐用户偏好分布（表3中DIIG+AU提升4.3%）。
        • Alignment guided by graded Item relations (AI)：分级对比学习（多模态相似 > 单模态相似 > 不相似）效果最佳（表4中AI优于SP/MP策略）。
      • 完整模型DA-MRS：整合所有模块后取得最优效果（表3最后一栏）。

    • 结论：DA-MRS的理论复杂度与LATTICE相当，低于MMSSL（附录A.3），实际训练时间和GPU内存消耗在可接受范围内。

创新点总结

  1. 多模态去噪：通过跨模态一致性构建更准确的物品关系图（DIIG）。
  2. 反馈去噪：提出基于多模态内容的概率生成模型（D-BPR），首次理论推导BPR损失的去噪形式。
  3. 细粒度对齐：用户偏好指导的任务对齐（AU）和分级物品关系对比（AI）超越传统item-level对齐。

DSGRec论文总结当前领域存在的问题 单一路径建模问题：现有方法通常只通过单一途径（用户交互信号或多模态信息）来建模用户兴趣，无法全面捕捉用户行为的复杂性。例如，用户选择商品可能既受相似用户推荐（需求信号）影响，也受商品本身的视觉/文本特征（偏好信号）影响。 信号协作不足问题：现有方法缺乏需求信号（CF信号）和偏好信号（MF信号）之间的细粒度协作。要么分开学习导致融合效果差，要么联合学习但可能产生优化方向冲突。 现有做法及局限现有方法主要分为两类： 分离学习法：先分别学习CF和MF的嵌入表示，再融合。问题：两个嵌入空间差异大，融合效果不佳。 联合学习法：在统一框架（如图神经网络）中同时学习CF和MF信号。问题：可能产生优化冲突。 这两类方法都无法适应真实场景中用户偏好可能由不同信号主导的情况。 本文提出的方法（DSGRec） LATTICE 等模型的多模态融合是 “混合式” 的（如将多模态特征作为节点属性融入交互图） 而 DSGRec 则是 “解耦式” 的：先独立提取 C-signals 和...
