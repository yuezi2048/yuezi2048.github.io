连接到后端服务器后，会得到一个会话的Session返回给前端

登录成功，会得到登录成功的Session，并对Session设置一些值（用户信息），返回给前端设置Cookie

前端收到相关命令后，则设置Cookie保存到浏览器内

后续前端请求，则会携带Cookie请求头请求

后端收到Cookie则会查找对应的Session，从对应的Session中取出响应的变量
