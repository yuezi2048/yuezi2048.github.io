六、数组、排序和查找

6.1 数组使用

为什么要数组：6只鸡，每只鸡的体重3kg,5kg,1kg,3.4kg,2kg,50kg ，问你平均体重怎么算？

  • 没有数组我们只能通过定义6个变量，引申到1w只鸡就不太合适了
  • 引申出数组，用来存放多个相同类型的数据
    • 注：数组是一种引用类型

定义有三种定义方法

  • 动态初始化
    • int a[] = new int[5];
    • 先声明int a[]，再分配空间 new int[5]
  • 静态初始化（已经知道具体的元素）
    • int a[] = {1, 2, 3};

细节

  • 数组是多个相同数据类型的集合，实现对多个数据统一管理
  • 数组的元素可以是基本类型，也可以是引用类型，不可混用
  • 数组创建后，有默认值
    • int 0， short 0, byte 0, long 0, float 0.0,double 0.0， char \u0000， boolean false， String null
  • 使用数组三个步骤：1.声明数组、开辟空间 2.数组赋值 3.使用数组
  • 数组的下标从0开始
  • 数组有下标限制，超出范围则会报越界异常
  • 数组本身是引用类型，数组型数据为对象Object

6.2 两种数组赋值机制*

两种赋值机制

  • 值拷贝（值传递）
    • int n1 = 2; int n2 = n1;
    • 一般是基本数据类型的赋值，此时赋值时，仅将栈中具体的值拷贝过来，后续n2变化以后n1不会变化
  • 地址拷贝（引用传递）
    • int[] arr1 = {1, 2, 3}; int arr2 = arr1;
    • 因为数组是引用类型，将数组类型赋给arr2，本质上是将arr1的引用地址拷贝给了arr2，而当数组的内容发生改变后，引用地址所指向的值的内容发生变化，arr1和arr2都会发生改变

6.5.1 二维数组使用

4种定义方式

动态初始化

  • int[][] arr = new int[2][3]
  • 先声明int[][]，再初始化new int[2][3]

列数不确定

  • int[][] arr = new int[3][];
    • 注意对二维数组中，每个元素需要重新开辟一块空间，此时只是对二维数组中一维数组进行了声明

静态初始化

  • int[]\[] arr = {{1}, {1, 2}, {1, 2, 3}};

  • String[] strs = new String[]{“a”, “b”, “c”};
