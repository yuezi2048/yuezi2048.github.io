Spring Boot Actuator жҳҜ Spring Boot жҸҗдҫӣзҡ„дёҖдёӘзӣ‘жҺ§е’Ңз®ЎзҗҶеҠҹиғҪзҡ„е·Ҙе…·гҖӮеҸҜд»Ҙзӣ‘жҺ§пјҡеҒҘеә·жғ…еҶөhealthгҖҒжҢҮж ҮmetricsгҖҒи·ҹиёӘиҜ·жұӮпјҢз®ЎзҗҶзҺҜеўғеҸӮж•°envзӯү

Actuator жҸҗдҫӣдәҶдё°еҜҢзҡ„з«ҜзӮ№пјҲEndpointsпјүпјҢз”ЁдәҺеұ•зӨәе’Ңж“ҚдҪңеә”з”Ёзҡ„еҶ…йғЁзҠ¶жҖҒгҖӮиҝҷдәӣз«ҜзӮ№еҸҜд»ҘйҖҡиҝҮHTTPгҖҒJMXзӯүж–№ејҸиҝӣиЎҢи®ҝй—®гҖӮе…·жңүз®ҖдҫҝжҖ§гҖҒеҸҜжү©еұ•жҖ§гҖҒе®үе…ЁжҖ§гҖҒжҳ“дәҺйӣҶжҲҗпјҲдёҫдҫӢпјҡйӣҶжҲҗSpring Securityй…ҚзҪ®и®ҝй—®жқғйҷҗпјҢдҝқйҡңдёҖдәӣж•Ҹж„ҹж–ӯзӮ№зҡ„е®үе…ЁжҖ§пјү

е®ғзҡ„дјҳеҠҝпјҡ

  • жҳ“дәҺйӣҶжҲҗпјҡејҖз®ұеҚіз”ЁпјҢз®ҖеҚ•й…ҚзҪ®еҚіеҸҜдҪҝз”ЁпјҢеҸҜд»ҘеҫҲе®№жҳ“ең°дёҺеӨ–йғЁзӣ‘жҺ§зі»з»ҹйӣҶжҲҗпјҢеҰӮ PrometheusгҖҒGrafana зӯүгҖӮ
  • дё°еҜҢзҡ„з«ҜзӮ№пјҡжҸҗдҫӣдәҶеӨ§йҮҸйў„е®ҡд№үзҡ„з«ҜзӮ№пјҢеҰӮеҒҘеә·жЈҖжҹҘпјҲ/healthпјүгҖҒеә”з”ЁдҝЎжҒҜпјҲ/infoпјүгҖҒеәҰйҮҸжҢҮж ҮпјҲ/metricsпјүзӯүгҖӮ
  • дё°еҜҢзҡ„дҝЎжҒҜпјҡжҸҗдҫӣдәҶе…ідәҺеә”з”ЁгҖҒзі»з»ҹгҖҒJVMгҖҒеҶ…еӯҳгҖҒзәҝзЁӢзӯүдё°еҜҢзҡ„дҝЎжҒҜгҖӮ
  • й«ҳеәҰеҸҜжү©еұ•пјҡеҸҜд»ҘиҮӘе®ҡд№үз«ҜзӮ№пјҢжү©еұ•жҲ–дҝ®ж”№зҺ°жңүзҡ„з«ҜзӮ№еҠҹиғҪгҖӮ
  • е®үе…ЁжҖ§пјҡж”ҜжҢҒз«ҜзӮ№зҡ„е®үе…Ёи®ҝй—®жҺ§еҲ¶пјҢеҸҜд»ҘйҖҡиҝҮй…ҚзҪ®йҷҗеҲ¶е“Әдәӣз«ҜзӮ№еҸҜд»Ҙе…¬ејҖи®ҝй—®гҖӮ

жү©еұ•зҹҘиҜҶ

Spring Actuator жҸҗдҫӣзҡ„з«ҜзӮ№

  • /actuator/healthпјҡжҳҫзӨәеә”з”Ёзҡ„еҒҘеә·дҝЎжҒҜпјҲй»ҳи®Өжғ…еҶөдёӢдјҡжЈҖжҹҘж•°жҚ®еә“иҝһжҺҘгҖҒж¶ҲжҒҜд»ЈзҗҶзӯүпјүгҖӮ
  • /actuator/infoпјҡжҳҫзӨәеә”з”Ёзҡ„иҮӘе®ҡд№үдҝЎжҒҜпјҢеҰӮзүҲжң¬гҖҒжҸҸиҝ°зӯүгҖӮ
  • /actuator/metricsпјҡжҳҫзӨәеә”з”Ёзҡ„еәҰйҮҸжҢҮж ҮпјҢеҰӮ JVM еҶ…еӯҳдҪҝз”Ёжғ…еҶөгҖҒеһғеңҫж”¶йӣҶдҝЎжҒҜгҖҒзәҝзЁӢдҝЎжҒҜзӯүгҖӮ
  • /actuator/loggersпјҡжҳҫзӨәе’Ңй…ҚзҪ®ж—Ҙеҝ—зә§еҲ«гҖӮ
  • /actuator/envпјҡжҳҫзӨәеә”з”Ёзҡ„зҺҜеўғй…ҚзҪ®дҝЎжҒҜпјҢеҢ…жӢ¬зі»з»ҹеұһжҖ§гҖҒзҺҜеўғеҸҳйҮҸгҖҒй…ҚзҪ®ж–Үд»¶еұһжҖ§зӯүгҖӮ
  • /actuator/beansпјҡжҳҫзӨә Spring еә”з”ЁдёҠдёӢж–ҮдёӯжүҖжңүзҡ„ Bean дҝЎжҒҜгҖӮ
  • /actuator/threaddumpпјҡжҳҫзӨәеә”з”Ёзҡ„зәҝзЁӢиҪ¬еӮЁдҝЎжҒҜгҖӮ
  • /actuator/httptraceпјҡжҳҫзӨәжңҖиҝ‘зҡ„ HTTP иҜ·жұӮе’Ңе“Қеә”и·ҹиёӘпјҲй»ҳи®Өжғ…еҶөдёӢеҸӘеӯҳеӮЁжңҖиҝ‘ 100 дёӘпјүгҖӮ
  • /actuator/mappingsпјҡжҳҫзӨәжүҖжңүзҡ„ URL жҳ е°„и·Ҝеҫ„гҖӮ
