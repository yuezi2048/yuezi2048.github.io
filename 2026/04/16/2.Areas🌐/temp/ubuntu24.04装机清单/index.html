ubuntu24.04装机清单

发表于2026-04-16|更新于2026-04-16|2.Areas🌐temp
|总字数:389|阅读时长:1分钟|浏览量:

Ubuntu24.04装机，24G内存很好用

前期工作：

  • 软件源
  • snap卸载+基于ppa重新安装firefox（可选）
  • flatpak软件库（推荐）：https://www.cnblogs.com/lwlnice/p/18263967
  • 星火应用商店（可选，个人使用时只能用终端版）
  • 铁豌豆安装必要软件（可选）
    • WPS365：测试 WPS 和 wayland 及其分数倍缩放率的契合度
    • Bilibili
    • Todesk 向日葵（有远程UI操作需求，建议X11显示模式兼容性拉满，不要用wayland）
    • 腾讯会议
    • QQ 微信
    • 音乐软件 Splayer
    • typora
    • vim
    • Steam
  • 显卡驱动（坑：注意ubuntu的内核版本+当前显卡的兼容版本，一般的话确定好能跑的内核的版本后最好就不要动了，老显卡可能要考虑降内核版本）
  • 局域网串流：moonlight + sunshine
  • 自搭中继服务器远程ssh连接/文件传输：RustDusk

软件清单：

  • 输入法：fcitx5 + Rime（中州韵）+ rime-ice词库增强 + 环境变量（https://zhuanlan.zhihu.com/p/508797663）
  • allacritty + 优化（可选）
  • clash: https://gitee.com/tools-repo/clash-for-linux-install
  • 开发工具
    • VSCode
    • Jetbrains: IDEA+Pycharm+WebStorm https://blog.csdn.net/JOJO_jiongjiong/article/details/123087307?spm=1001.2014.3001.5506

GNOME优化工作：

  • 壁纸：https://haowallpaper.com/homeViewLook/16660540411137408
  • 插件：https://extensions.gnome.org
    • Blur my Shell
    • Compact Top Bar
    • Dash to Panel
    • Extension List
    • Gradient Top Bar
    • Hide Top Bar
    • Input Method Panel
    • One Window Wonderland
    • Show Desktop Button
    • System Monitor
    • User Themes
  • GRUB加载页面优化（可选）：https://www.gnome-look.org/p/2076542

GNOME动态壁纸（可选）：https://ivonblog.com/posts/gnome-hidamari/
nemo文件管理器（可选）：https://gist.github.com/nenekodev/78744e3de3074d58d258bd1cbd19141e
安卓模拟器（可选）：麟卓卓懿
虚拟机（可选）
