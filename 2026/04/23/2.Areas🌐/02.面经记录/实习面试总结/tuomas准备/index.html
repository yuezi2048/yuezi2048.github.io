线上面试极速复习提纲（按优先级排序）

核心原则：100%贴合岗位职责+任职要求，优先深挖简历亮点，优先搞定面试占比70%以上的核心考点，适配今晚9点-次日11点的备考时间窗口。

P0 最高优先级（今晚必须全部吃透，面试占比70%，建议耗时3.5小时）

一、简历项目深度复盘（重中之重，面试官80%的深挖问题来自这里）

必须为每个问题准备**「结论+原理+项目实操案例+数据结果」**的标准化话术，做到脱口而出，重点绑定岗位「全栈开发、AI编程实践」核心要求。

九华云SaaS财税发票管理平台（校企合作真实商业项目）

必背核心问题：

  1. 项目业务背景、你的核心职责、团队分工、开发周期，你作为负责人&全栈主导，核心解决了甲方什么核心痛点？
  2. 20+精准索引设计：联合索引的字段顺序怎么定的？怎么规避回表？怎么验证优化效果？除了索引，还有哪些动作让100w数据查询从2s降到200ms内？
  3. 多对多联动场景：为什么用冗余字段+逻辑外键，不用物理外键和JOIN？怎么保证冗余字段的数据一致性？遇到过事务失效吗？怎么解决的？
  4. 自研Spring Boot Starter实现Excel批量导入：Starter自动配置的核心原理？自定义注解的实现逻辑？大批量导入怎么规避OOM？
  5. 并发发号方案：为什么不用雪花算法/UUID？「预生成号段+定时预热+内存加载+原子分配」全流程怎么实现？服务重启怎么保证号段连续？分布式部署怎么适配？
  6. 前端全栈部分：Vue3+TS封装了哪些通用组件？怎么实现样式隔离和跨模块复用？快速录入交互的设计思路，80%的效率提升怎么测算的？
  7. 【岗位核心绑定】这个项目里，你用Claude Code做了哪些事？具体在哪个环节提升了效率？AI生成的代码踩过什么坑？怎么校验和解决的？

凌云团队相册管理平台（独立全栈开发）

必背核心问题：

  1. 「Caffeine本地+Redis分布式」二级缓存方案：为什么这么设计？缓存一致性怎么保证？缓存击穿/穿透/雪崩怎么解决？双策略热点识别的阈值怎么定的？
  2. ShardingSphere-JDBC分表：为什么按spaceId分片？自定义分片算法的逻辑？分表后怎么处理分页/聚合查询？千万级数据查询稳定30ms内的核心优化点？
  3. WebSocket+Disruptor协同编辑：为什么用Disruptor无锁队列？怎么保证操作的顺序性和最终一致性？50ms内的跨端同步怎么优化的？
  4. 【岗位核心绑定】这个独立项目中，AI工具帮你解决了哪些技术难点？比如分表方案设计、协同编辑逻辑开发、性能调优等，举具体案例。

二、岗位硬性要求核心考点（100%匹配任职要求，缺项必须紧急补全）

AI辅助编程工具（岗位职责核心，任职要求硬性门槛）

必背内容：

  1. 你熟练使用Claude Code的具体落地场景：需求拆解、代码生成、调试排错、SQL优化、单元测试、代码重构、文档生成，每个场景绑定1个项目中的真实案例。
  2. 你沉淀的AI编程最佳实践：怎么写Prompt能生成高质量代码？怎么规避AI代码的逻辑漏洞、安全问题、性能隐患？你的AI辅助开发全流程规范。
  3. 踩坑案例：AI生成代码出现过什么问题？你怎么发现、校验和修复的？
  4. 岗位提及工具的基础认知：快速了解Cursor、Trae、OpenClaw的核心特点、适用场景，能说出和Claude Code的差异，对应岗位职责的「AI工具研究」要求。
  5. 团队推广思路：怎么把AI辅助开发的经验沉淀成规范，给团队做内部分享，提升整体研发效率，对应岗位职责的「技术调研与内部培训」要求。

Java核心+Spring生态（任职要求熟练掌握，简历核心亮点）

  • 必背重点（只看和项目相关的，不泛学）：
    1. Spring核心：IOC/AOP实现原理、事务ACID、事务传播机制、事务失效的7种核心场景及解决方案。
    2. Spring Boot Starter自动配置原理（你自研过Starter，必须吃透全流程）。
    3. 多线程并发：线程池核心7大参数、你项目中线程池的设计思路、CAS/AQS原理、synchronized与ReentrantLock的区别。
    4. 集合框架：HashMap/ConcurrentHashMap源码核心原理、1.7与1.8的差异。

MySQL（任职要求熟练掌握，简历有大量优化案例）

  • 必背重点：
    1. 索引核心：B+树结构、聚簇/非聚簇索引、联合索引最左匹配原则、回表与覆盖索引、索引失效的常见场景。
    2. 事务与锁：MVCC实现原理、隔离级别、行锁/表锁/间隙锁/临键锁的适用场景。
    3. SQL优化全流程：慢查询排查步骤、你项目中的优化方法论，必须和项目案例绑定。

前端+node.js（任职要求硬性门槛，全栈岗位职责核心）

  • 必背重点（简历薄弱项，紧急补全）：
    1. 前端：ES6+核心语法（Promise、async/await、箭头函数、解构赋值、模块化）、Vue3核心（ref/reactive区别、组合式API、响应式原理）、TypeScript基础类型与接口。
    2. node.js：核心模块（fs、path、http）、异步编程模式、Express/Koa中间件原理、CommonJS与ES Module的区别，准备好「有扎实JS/TS基础，通过AI工具快速学习node.js，已完成XX小demo」的话术，规避短板。

P1 次高优先级（次日上午重点准备，面试占比20%，建议耗时2.5小时）

一、简历技能点补充深挖

  1. Redis：五大数据类型适用场景、持久化机制（RDB/AOF区别）、分布式锁实现原理、缓存三大问题的完整解决方案，绑定相册项目二级缓存案例。
  2. 算法与SQL：快速刷LeetCode高频TOP20简单题（链表、二叉树、数组）、高频SQL题（联表查询、分组聚合、索引优化），应对手撕环节。
  3. 工程化：Git核心协作命令、冲突解决、分支管理策略、阿里巴巴编码规范核心要求、Linux常用排查命令。
  4. 数据中台基础认知：快速了解数据中台的核心定位、核心能力，梳理你的项目经验怎么迁移到数据中台产品的全栈开发中。

二、自我介绍与行为面试题标准化准备

  1. 自我介绍：打磨1分钟极速版、3分钟完整版，全程紧扣岗位要求，核心结构：
    学历背景（硕士+前10%绩点）→ 核心技术栈（Java全栈+AI工具使用）→ 核心项目亮点（真实商业项目负责人经验+性能优化成果）→ 求职动机（对全栈技术+AI工程化的热情，匹配岗位核心职责）。
  2. 高频行为面试题必背话术：
    • 你对这个岗位的理解是什么？为什么应聘这个岗位？
    • 项目中遇到的最大技术挑战是什么？你怎么拆解和解决的？
    • 你怎么看待AI编程工具对研发工程师的影响？
    • 你的职业规划是什么？实习期间的核心目标是什么？
    • 团队协作中遇到过意见分歧吗？你怎么处理的？
    • 你最快的到岗时间？能保证的实习时长？

P2 兜底优先级（快速过一遍，面试占比10%，碎片时间浏览）

  1. 计算机基础高频考点：TCP三次握手/四次挥手、HTTP/HTTPS原理、进程与线程的区别、操作系统内存管理。
  2. 简历提及的拓展知识点：RabbitMQ核心场景与消息可靠性保证、ShardingSphere进阶特性、机器学习/大模型应用基础概念。
  3. 岗位提及的Trae、OpenClaw等AI工具的基础功能，快速了解核心定位即可。

备考时间规划（精准到小时）

时间段 核心任务 优先级
今晚21:00-22:30 两个项目全量问题复盘，写好标准化话术 P0
今晚22:30-23:30 AI辅助编程工具全量内容准备，绑定岗位要求 P0
今晚23:30-00:30 Java+MySQL+前端+node.js核心考点攻坚 P0
00:30-次日7:30 保证睡眠，维持面试状态（严禁熬夜） -
次日7:30-8:30 自我介绍打磨+行为面试题话术定稿 P1
次日8:30-10:00 算法/SQL高频题+Redis+数据中台基础复习 P1
次日10:00-10:30 设备调试+自我介绍+核心亮点模拟复述，调整状态 -

面试核心注意事项

  1. 所有回答优先绑定岗位3个核心职责（全栈开发、AI编程实践、技术调研与内部分享），把你的经历往岗位需求上靠。
  2. 技术问题回答公式：先给结论→再讲核心原理→最后用你的项目案例佐证，避免空泛背书。
  3. 面对node.js等简历薄弱项，不回避不撒谎，主动突出「扎实的前端基础+AI工具赋能的快速学习能力+可落地的学习计划」，匹配岗位的学习能力要求。
  4. 全程突出你的核心优势：硕士学历+扎实的计算机基础+真实商业项目负责人经验+全栈落地能力+AI工具实战经验，完全匹配岗位任职要求。
