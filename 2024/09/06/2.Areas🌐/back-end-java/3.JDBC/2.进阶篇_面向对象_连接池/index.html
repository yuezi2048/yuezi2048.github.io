1.1 ORMжҖқжғі

  • ORMпјҲObject Relational MappingпјүжҖқжғіеҚідёәеҜ№иұЎеҲ°е…ізі»ж•°жҚ®еә“зҡ„жҳ е°„пјҢйқўеҗ‘еҜ№иұЎдёҺж•°жҚ®еә“иЎЁзҡ„жҰӮеҝөеҜ№еә”иө·жқҘдәҶ
    • й—®йўҳпјҡJDBCж“ҚдҪңж—¶пјҢж•°жҚ®еә“зҡ„дёҖиЎҢе®Ңж•ҙзҡ„ж•°жҚ®пјҢдј еҲ°дәҶJavaеҚҙжҳҜдёҖдёӘдёӘеҸҳйҮҸпјҢдёҚеҲ©дәҺз»ҙжҠӨе’Ңз®ЎзҗҶ
    • и§ЈеҶіпјҡдёҖеј иЎЁеҜ№еә”дёҖдёӘзұ»пјҢдёҖиЎҢж•°жҚ®еҜ№еә”дёҖдёӘеҜ№иұЎпјҢдёҖеҲ—еҜ№еә”еҜ№иұЎзҡ„еұһжҖ§пјҢеҚіе°Ҷж•°жҚ®еӯҳеӮЁеңЁдёҖдёӘе®һдҪ“зұ»йҮҢ
  • зҺ°еңЁжҲ‘д»¬дҪҝз”ЁJDBCе®һзҺ°жүӢеҠЁORMпјҢеҗҺз»ӯжңүORMжЎҶжһ¶пјҢеҰӮMyBatisгҖҒJPAзӯү

жөӢиҜ•

1.2 дё»й”®еӣһжҳҫ

  • й—®йўҳпјҡеңЁж•°жҚ®еә“дёӯж–°еўһж•°жҚ®пјҢдё»й”®еҲ—иҮӘеҠЁеўһй•ҝпјҢеҸҜд»ҘеңЁиЎЁдёӯзӣҙжҺҘзңӢеҲ°пјҢдҪҶJavaзЁӢеәҸеҸӘиғҪеҫ—еҲ°еҸ—еҪұе“Қзҡ„иЎҢж•°пјҢж— жі•еӣһжҳҫеҫ—еҲ°иЎЁзҡ„дё»й”®еҖј
  • и§ЈеҶіпјҡ
    • йңҖиҰҒеңЁйў„зј–иҜ‘SQLж—¶пјҢе‘ҠзҹҘSQLиҝ”еӣһж–°еўһзҡ„дё»й”®еҲ—пјҢж–№жі•пјҡеҠ дёҠеҸӮж•°Statement.RETURN_GENERATED_KEYS
    • еңЁжҲҗеҠҹжҸ’е…Ҙж•°жҚ®еҗҺпјҢдҪҝз”ЁpreparedStatement.getGeneratedKeys()ж–№жі•иҝ”еӣһдё»й”®з»“жһңйӣҶпјҢе№¶иөӢеҖјз»ҷеҜ№еә”еҜ№иұЎзҡ„idзӯүеұһжҖ§дёҠ
  • еә”з”Ёпјҡз”ЁжҲ·жүӢжңәеҸ·жіЁеҶҢжҲҗеҠҹеҗҺпјҢйңҖиҰҒиҝӣдёҖжӯҘеЎ«еҶҷе§“еҗҚзӯүдҝЎжҒҜпјҢJavaйңҖиҰҒеҫ—еҲ°дё»й”®жӣҙж–°з”ЁжҲ·зҡ„зӣёе…іи®°еҪ•

1.3 жү№йҮҸж“ҚдҪң

  • й—®йўҳпјҡжҸ’е…ҘеӨҡжқЎж•°жҚ®ж—¶пјҢдёҖжқЎдёҖжқЎеҸ‘йҖҒз»ҷж•°жҚ®еә“жү§иЎҢпјҢж•ҲзҺҮдҪҺдёӢпјҒ

  • и§ЈеҶіпјҡйҖҡиҝҮжү№йҮҸж“ҚдҪңпјҢеҸҜд»ҘжҸҗеҚҮеӨҡж¬Ўж“ҚдҪңж•ҲзҺҮпјҒ

    • JDBCй»ҳи®ӨдёҚж”ҜжҢҒжү№йҮҸпјҢйңҖиҰҒеңЁиҝһжҺҘж—¶еҠ дёҠеҸӮж•°rewriteBatchedStatements=true
    • жү§иЎҢзҡ„ж–№жі•д»ҺexecuteUpdate()ж”№дёәaddBatch()пјҢжңҖеҗҺи°ғз”ЁexecuteBatch()ж–№жі•е®һзҺ°

дәҢгҖҒиҝһжҺҘжұ 

2.1 дёәд»Җд№ҲиҰҒиҝһжҺҘжұ 

  • й—®йўҳпјҡ
    • з”ЁжҲ·жҜҸж¬Ўж“ҚдҪңж•°жҚ®еә“йғҪиҰҒиҺ·еҸ–ж–°иҝһжҺҘпјҢдҪҝз”Ёе®ҢжҜ•еҗҺе°ұйҮҠж”ҫпјҢйў‘з№ҒеҲӣе»әе’Ңй”ҖжҜҒиҝһжҺҘдјҡйҖ жҲҗиө„жәҗжөӘиҙ№ вҖ“> жңүдёҖдёӘд»ЈзҗҶеҰӮжһңеҸҜд»ҘеӨҚз”Ёз”ЁжҲ·йҮҠж”ҫзҡ„иҝһжҺҘе°ұеҘҪдәҶ
    • иҝһжҺҘзҡ„ж•°йҮҸдёҚзЎ®е®ҡпјҢжңҚеҠЎеҷЁеҸҜиғҪиҙҹиҪҪиҝҮйҮҚ вҖ“> йңҖиҰҒжңүз®ЎзҗҶиҖ…
  • и§ЈеҶіпјҡдҪҝз”ЁиҝһжҺҘжұ 
    • иҝһжҺҘжұ жҳҜж•°жҚ®еә“иҝһжҺҘеҜ№иұЎзҡ„зј“еҶІеҢәпјҢйҖҡиҝҮй…ҚзҪ®пјҢеҸҜд»Ҙи®©иҝһжҺҘжұ еҲӣе»әиҝһжҺҘгҖҒз®ЎзҗҶиҝһжҺҘгҖҒйҮҠж”ҫиҝһжҺҘ
    • йў„е…Ҳе°Ҷж•°жҚ®еә“зҡ„иҝһжҺҘж”ҫе…ҘиҝһжҺҘжұ пјҢз”ЁжҲ·еңЁиҜ·жұӮж—¶пјҢзӣҙжҺҘйҖҡиҝҮиҝһжҺҘжұ иҺ·еҸ–иҝһжҺҘпјҢе®ҢжҜ•еҗҺе°ҶиҝһжҺҘеҪ’иҝҳз»ҷжұ пјҢйҒҝе…ҚдәҶйў‘з№Ғзҡ„еҲӣе»әе’Ңй”ҖжҜҒпјҢе№¶ж”№е–„еҲӣе»әзҡ„ж•ҲзҺҮ
    • иҝһжҺҘжұ еҲҶй…ҚиҝһжҺҘзҡ„и§„еҲҷ
      • еҰӮжһңжұ ж— иҝһжҺҘеҸҜз”ЁдҪҶжІЎжңүеҲ°иҫҫдёҠйҷҗпјҢеҲҷж–°е»әиҝһжҺҘ
      • еҰӮжһңжұ еҲ°иҫҫдёҠйҷҗпјҢз”ЁжҲ·зӯүеҫ…пјҢеҸҜи®ҫзҪ®и¶…ж—¶ж—¶й—ҙпјҲ13дәҝдәәж°‘жү“е®ўжңҚзҡ„дҫӢеӯҗпјү

й…ҚзҪ® зјәзңҒ иҜҙжҳҺ
