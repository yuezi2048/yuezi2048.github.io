JDK核心： javac：编译器，将源代码编译为字节码class文件 java：运行java命令，通过JVM来解释并执行字节码文件 javadoc：生成doc文档 jar：创建和管理JAR文件的工具，多个class文件打包成一个文件便于分发管理 jdb：java调试工具，在命令行中调试java程序，断点调试、变量查看等 性能监控： jps：查看java中的进程，linux会用到 jstack：生成堆栈信息工具，分析死锁、线程问题 jmap：内存映射，生成堆存储（head dump）文件，分析内存泄漏，垃圾回收优化 jstat：JVM统计工具，配合jmap可以分析堆存储文件，了解内存情况 jconsole：图形化的JVM监控，监控应用程序内存、线程、类加载情况 jvisualvm：性能分析工具，支持堆、线程、GC监控，以及内存CPU分析 诊断工具： jinfo：查看修改正在运行的JVM参数，动态调优和调整JVM行为 jstatd：远程JVM监控，通过网络远程监控JVM，用在分布式性能监控 拓展： 高级调试和性能调优工具 Java Mission...
